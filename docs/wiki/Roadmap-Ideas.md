# Roadmap Ideas

This page collects ideas that are not fully built yet.

## Online Multiplayer

Possible future multiplayer options:

- Join by room code.
- Friend invite link.
- Turn sync through a free backend service.
- A simple lobby screen.
- Optional bot fallback if a player leaves.

Since some home internet setups do not support port forwarding, a hosted backend or peer service would be better than running a home server.

## Leveling System

Possible leveling ideas:

- XP for finishing matches.
- Bonus XP for comeback wins.
- Cosmetic unlocks.
- Profile titles.
- Long-term coin or item stats.

## More Items

Possible item directions:

- More opponent disruption.
- More risk-reward items.
- Items that affect both players.
- Items that modify shop offers.
- Items that change the next round setup.

Peek-style information items are being avoided for now because they can feel weak unless the game gives the player a strong reason to use that information.

## Casino Table Upgrade

Visual upgrade ideas:

- Richer felt texture.
- Better table edges.
- Chip and coin accents.
- More polished item icons.
- Stronger card flip and slide animation.
- Cleaner win and shop transitions.
