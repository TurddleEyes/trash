# Game Modes

Trash Cards currently has two game modes.

## Classic Trash

Classic Trash is the clean version of the game.

- No coins.
- No items.
- No shop.
- Same round structure.
- The winner of each round needs one fewer card next round.
- Win a round while needing only 1 card to win the match.

Classic Trash is best when you want the pure card game.

## Trash: Crown Debt

Trash: Crown Debt keeps the same card rules, then adds:

- Coins after every round.
- A between-round shop.
- Items with common, uncommon, rare, and legendary tiers.
- A 2-item inventory limit.
- A forced discard rule for winners holding 2 items.
- The legendary Debt of the Crown comeback item.

Crown Debt is best when you want the main game to feel more strategic, chaotic, and replayable.

## Main Difference

| Feature | Classic Trash | Trash: Crown Debt |
| --- | --- | --- |
| Matching card slots | Yes | Yes |
| Rounds from 10 down to 1 | Yes | Yes |
| Coins | No | Yes |
| Shop | No | Yes |
| Items | No | Yes |
| Forced item discard after winning with 2 items | No | Yes |
| Legendary comeback item | No | Yes |
