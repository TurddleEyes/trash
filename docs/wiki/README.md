## What Is Trash Cards?

Trash Cards is a turn-based card game where each player tries to fill numbered slots with matching cards.

- Ace goes in slot 1.
- Cards 2 through 10 go in their matching slots.
- Jacks, queens, and kings are trash unless an item changes that.
- When a round is won, the winner needs one fewer slot next round.
- The match ends when a player wins a round while only needing 1 card.

## Current Versions

The game currently has two modes:

- **Classic Trash**: The original race from 10 cards down to 1.
- **Trash: Crown Debt**: Adds rounds, coins, shop items, rare effects, and comeback tools.

This wiki matches the current browser build around release `v0.2.3`.
