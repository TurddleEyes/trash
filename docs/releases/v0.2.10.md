# v0.2.10 - Multiplayer Timers

## Highlights

- Added shared online Crown Debt shops so both players choose from the same offers.
- Online shops now wait until both players are ready or their shop timers expire.
- Added online turn timers that reset after draw, place, discard, and item actions.
- Added inactivity match endings with 5% saved-point penalties for repeated timeouts.

## Notes

- Timeout penalties are applied by the public Worker and only the room host can submit the penalty event.
- The current account system stores leaderboard value through stats and coins, so the penalty is applied by reducing saved coins based on 5% of the player's leaderboard score.
