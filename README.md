# Trash Cards

Trash Cards is a browser card game based on the classic game Trash/Garbage.

Play it here:

[https://turddleeyes.github.io/trash/](https://turddleeyes.github.io/trash/)

## Game Modes

- **Classic Trash**: A clean race from 10 cards down to 1.
- **Trash: Crown Debt**: Adds rounds, coins, shop items, and the legendary Debt of the Crown comeback item.
